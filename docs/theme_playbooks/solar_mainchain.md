# 光伏主链主题认知卡

## 市场通常在交易什么

- 市场在交易的是装机预期、产业链价格、去库节奏和龙头盈利修复，而不是泛“新能源”口号。
- 光伏主链更偏产业链供需和价格周期，和钙钛矿这类新技术线不是同一种首页写法。

## 常见正向驱动

- 国内外装机需求改善、组件和逆变器出货回暖、库存去化加快。
- 产业链价格企稳，龙头利润率和现金流预期修复。

## 常见反向风险

- 需求没坏，但供给过剩和价格战仍压制盈利。
- 市场先交易反转预期，后续发现去库和利润率恢复慢于预期。

## 首页应优先联想到的宏观 / 产业变量

- 装机、招标、组件价格、逆变器出货、库存、海外需求和政策。

## 典型传导链

- 装机 / 去库改善 -> 组件与逆变器出货修复 -> 龙头利润率预期改善 -> 光伏板块估值修复。

## 常见所处阶段

- 常见于价格战后的低位修复或景气拐点预期阶段。

## 轮动与拥挤度

- 一旦景气拐点预期强化，容易快速从低位板块切成拥挤修复交易。

## 证伪信号

- 产业链价格继续下探、库存难去、装机和出货验证不及预期。

## 不能误写成直接催化的东西

- “新能源长期空间大”不能直接写成当天催化。
- 行业政策支持不能替代价格和盈利验证。

## 写首页时应避免的模板化表达

- 不要把光伏主链写成“新能源反弹”就结束。
- 要点明今天是在交易需求修复、价格企稳，还是单纯估值反弹。
