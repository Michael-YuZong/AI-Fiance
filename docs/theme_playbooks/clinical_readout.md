# 创新药临床催化主题认知卡

## 市场通常在交易什么

- 市场在交易的是临床读数、关键数据、审批节点和产品管线价值重估。
- 这条线更偏单点验证和事件驱动，波动通常比宽板块更大。

## 常见正向驱动

- 临床数据超预期、关键终点达标、审批推进或适应症拓展。
- 管线价值被重新定价，带动个股和小范围主题弹性。

## 常见反向风险

- 数据不及预期，或者节点推迟。
- 个股事件很强，但很难扩散成整个板块趋势。

## 首页应优先联想到的宏观 / 产业变量

- 临床节点、审批、适应症、关键终点、管线价值和成长风格。

## 典型传导链

- 临床 / 审批节点 -> 个股风险偏好跳升 -> 同主题板块出现小范围扩散。

## 常见所处阶段

- 更常处在事件前预期和事件后兑现的快速切换阶段。

## 轮动与拥挤度

- 事件型拥挤最强，前后波动都大，最怕预期打太满。

## 证伪信号

- 数据推迟、终点不及预期、审批节奏变慢。

## 不能误写成直接催化的东西

- “管线丰富”不能自动写成今天临床催化。
- 事件预期不能替代真实读数。

## 写首页时应避免的模板化表达

- 不要把临床读数写成泛医药修复。
- 不要把个股级事件直接扩大成整个板块必涨。
