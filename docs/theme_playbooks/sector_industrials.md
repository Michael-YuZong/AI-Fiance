# 工业行业认知卡

## 市场通常在交易什么

- 市场在交易的是资本开支、制造业景气、出口订单和基础设施投资，而不是单纯的“设备更新”口号。
- 机械、军工、自动化、运输装备和工程设备都属于工业，但顺风变量不完全相同。

## 常见正向驱动

- PMI 与新订单改善、出口修复、设备更新、基础设施投资和高端制造景气上行。
- 龙头订单、产能利用率和交付节奏被验证。

## 常见反向风险

- 宏观和订单预期不错，但盈利兑现慢、价格竞争强。
- 行业内部分化很大，低景气子方向会拖累整体判断。

## 首页应优先联想到的宏观 / 产业变量

- PMI、新订单、资本开支、出口、基础设施投资、产能利用率和交付周期。

## 典型传导链

- 宏观与资本开支改善 -> 订单和排产抬升 -> 龙头利润率与现金流验证 -> 行业内部分化扩散。

## 常见所处阶段

- 更常见的是“预期修复 -> 订单验证 -> 业绩兑现 -> 去伪存真”，而不是只靠事件口号持续抬估值。

## 轮动与拥挤度

- 工业常在基建链、自动化、高端装备和军工之间切换；不要把所有工业子方向写成同一条主线。

## 证伪信号

- 如果新订单、出口和资本开支没有接力，或龙头订单落地不及预期，这条行业逻辑就不能继续按扩散主线去写。

## 不能误写成直接催化的东西

- “设备更新 / 高端制造是国家方向”不能直接写成当天催化。
- 行业长期空间不能替代订单、交付和利润率验证。

## 写首页时应避免的模板化表达

- 不要只写“工业景气修复”。
- 要说明今天交易的是基建、出口、自动化还是军工链订单。
