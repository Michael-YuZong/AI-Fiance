# 创新药主题认知卡

## 市场通常在交易什么

- 创新药管线价值、BD 授权、临床读数、医保与出海预期。
- 市场通常先交易“估值修复 + 管线预期”，再等更硬的业绩和临床验证。
- 指数和个股经常分化，龙头与普通跟涨标的的逻辑并不一样。
- 创新药方向兼具成长、政策和风险偏好属性，容易受风格切换影响。

## 常见正向驱动

- 临床数据、BD 授权、海外合作、医保和审评进度改善。
- 行业政策边际改善，资金重新愿意给成长医药估值。
- 出海逻辑强化，龙头产品或平台能力获得外部验证。
- 大市值龙头修复带动板块 risk-on，ETF 或主题篮子先得到估值修复。

## 常见反向风险

- 管线预期很好，但兑现周期太长，短线容易回吐。
- 风格切回防御或资源后，成长医药承压。
- 医保、集采、审批、竞争格局等政策变量仍然会反复扰动情绪。
- 个股层面如果缺少临床或 BD 验证，容易只剩“便宜和想象力”。

## 首页应优先联想到的宏观 / 产业变量

- 成长风格顺逆风、海外利率、港股 / A股医药风险偏好。
- 临床读数、授权出海、医保谈判、审批进度。
- 龙头公司与 ETF / 板块之间，是在交易龙头验证，还是在交易整体估值修复。
- 创新药不是纯政策题材，也不是纯防守板块，要写清现在市场到底在交易哪一面。

## 典型传导链

- 临床 / BD / 出海验证 -> 龙头风险偏好修复 -> 板块估值修复和主题 ETF 弹性释放。

## 常见所处阶段

- 常处在“估值修复先行、临床和授权再验证”的早中期阶段。

## 轮动与拥挤度

- 风格偏成长时容易被快速抱团；一旦成长回撤，创新药通常先被兑现高弹性部分。

## 证伪信号

- 临床或 BD 迟迟没有新增验证、医保 / 竞争格局扰动加大，说明这条线仍停留在想象阶段。

## 不能误写成直接催化的东西

- “创新药长期空间大”不能直接写成当天催化。
- 行业估值修复不能替代个股临床、授权和业绩验证。
- 医药板块热度和讨论度不能直接写成产品成功概率提升。
- 对 CRO / 创新药行业常识的理解，不能直接偷渡成个股级正向事件。

## 写首页时应避免的模板化表达

- 不要只写“医药估值低位修复”，要写清是在交易临床、BD、出海还是风格切换。
- 不要把创新药和 CRO 混成一个逻辑。
- 不要把“政策改善”写成模糊口号。
- 不要把长期逻辑自动翻译成今天可以直接做。
