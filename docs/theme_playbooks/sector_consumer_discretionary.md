# 可选消费行业认知卡

## 市场通常在交易什么

- 市场在交易的是居民收入预期、场景恢复、消费升级和可选支出的弹性，而不是所有消费都会一起好。
- 汽车、家电、白酒、旅游、免税和餐饮供应链虽然都可归入可选消费，但它们受收入、地产和出行链条影响的方式不一样。

## 常见正向驱动

- 居民信心回升、促消费政策、节假日出行恢复和以旧换新等需求刺激。
- 龙头动销、库存和价格带改善，带动估值修复。

## 常见反向风险

- 收入和就业预期偏弱，可选支出恢复不及市场预期。
- 价格战、库存压力和需求分化让行业只剩情绪修复。

## 首页应优先联想到的宏观 / 产业变量

- 社零、出行、地产后周期、促销力度、库存、居民收入与就业预期。

## 典型传导链

- 居民预期 / 政策刺激改善 -> 客流和动销回暖 -> 龙头利润率与现金流修复 -> 板块估值扩散。

## 常见所处阶段

- 更常见的是“预期修复 -> 数据验证 -> 龙头分化 -> 子行业扩散”，而不是全行业同步右上。

## 轮动与拥挤度

- 可选消费内部通常先由最确定的子行业领涨，再扩散到弹性更高的方向；高端与大众、内需与出海也会明显分化。

## 证伪信号

- 如果客流、社零和库存没有同步改善，或者地产和收入预期继续走弱，这条行业逻辑就不能继续往乐观方向写。

## 不能误写成直接催化的东西

- “消费复苏预期”不能直接写成当天催化。
- 节假日和政策口号不能替代销量、库存和利润率验证。

## 写首页时应避免的模板化表达

- 不要把白酒、家电、免税、汽车和餐饮写成一句“可选消费回暖”。
- 要说明今天到底是谁在领涨，以及交易的是场景恢复、收入预期，还是估值修复。
