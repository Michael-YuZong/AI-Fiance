# 黄金 / 有色资源主题认知卡

## 市场通常在交易什么

- 黄金更常交易避险、美元与利率、央行购金和地缘溢价；有色更常交易全球需求、库存周期和工业金属价格。
- 两者都能归到资源链，但黄金偏防守和避险，有色偏顺周期和工业需求。
- 当市场写资源主题首页时，首先要区分今天在涨的是“黄金逻辑”还是“工业金属逻辑”。

## 常见正向驱动

- 黄金：地缘风险、美元走弱、实际利率回落、央行持续增持。
- 有色：制造业修复、库存去化、海外基建与 AI 电力链需求、供给瓶颈。
- 风格从高弹性成长切向资源和商品链。
- 龙头资源股现金流和分红同步改善。

## 常见反向风险

- 金价或铜铝价格高位震荡，边际驱动开始钝化。
- 地缘溢价回落，黄金股回吐快于金价本身。
- 资源价格还稳，但股票已经先把乐观预期 price in。
- 黄金和有色虽然都叫资源，但如果混写，首页判断会失真。

## 首页应优先联想到的宏观 / 产业变量

- 黄金：美元、实际利率、央行购金、地缘风险。
- 有色：铜铝价格、库存、制造业修复、电网和 AI 电力链需求。
- 当前市场是在交易避险、防守，还是全球需求和顺周期。

## 典型传导链

- 黄金：美元 / 实际利率 / 央行购金 -> 金价 -> 黄金股与黄金 ETF 的避险溢价。
- 有色：全球需求 / 库存去化 -> 铜铝等工业金属价格 -> 资源股盈利与估值修复。

## 常见所处阶段

- 更常处在“避险升温后的高位震荡”或“全球需求修复下的顺周期扩散”阶段。

## 轮动与拥挤度

- 黄金常在风险偏好回落时被资金抱团；有色常在顺周期扩散和电力链叙事强化时拥挤度抬升。

## 证伪信号

- 黄金：美元和实际利率同步回升、地缘溢价回落。
- 有色：金属价格回落且库存重新累积，顺周期修复被证伪。

## 不能误写成直接催化的东西

- “资源逻辑仍在”不能自动写成当天催化。
- 金价或铜价高位不能自动等于资源股 / ETF 现在可追。
- 地缘风险叙事不能替代趋势确认和风险收益比。

## 写首页时应避免的模板化表达

- 不要把黄金和有色写成同一句“受益于资源价格上涨”。
- 不要只说“避险升温”或者“顺周期修复”，要选清楚今天是哪一种。
- 不要忽略资源股高位震荡时的拥挤风险。
