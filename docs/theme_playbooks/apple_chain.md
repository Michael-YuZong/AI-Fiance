# 苹果链主题认知卡

## 市场通常在交易什么

- 市场在交易的是 iPhone / iPad / Mac / Vision Pro 等新品周期、出货预期和供应链份额变化。
- 苹果链更像全球消费电子周期和龙头客户订单映射，不只是一个短线事件题材。

## 常见正向驱动

- 新品发布、备货周期、出货预期改善、零部件份额提升。
- 海外消费电子需求改善，或者新产品带来新一轮创新周期。

## 常见反向风险

- 新品周期不及预期，备货和出货口径边际转弱。
- 主题经常先交易预期，兑现时容易回落。

## 首页应优先联想到的宏观 / 产业变量

- 新品周期、备货、出货、消费电子景气、美元和全球消费需求。

## 典型传导链

- 新品周期 / 备货 -> 关键零部件和组装环节订单 -> 苹果链供应商盈利预期改善。

## 常见所处阶段

- 常见于事件前备货交易和事件后出货验证阶段。

## 轮动与拥挤度

- 在消费电子修复和大客户新品周期叠加时最容易成为抱团方向。

## 证伪信号

- 备货和出货预期走弱，或者新品创新幅度不足以带来新一轮需求拉动。

## 不能误写成直接催化的东西

- “苹果要出新品”不能直接写成当天催化。
- 消费电子长期空间不能替代供应链订单验证。

## 写首页时应避免的模板化表达

- 不要把苹果链写成纯事件脉冲，也不要忽略全球消费电子周期。
