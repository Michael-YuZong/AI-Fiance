# 材料行业认知卡

## 市场通常在交易什么

- 市场在交易的是商品价格、供给纪律、库存周期和价差，而不是一个统一的“材料板块走强”。
- 黄金、有色、化工、建材和新材料虽然都在材料大类里，但定价逻辑差异很大。

## 常见正向驱动

- 商品价格上行、供给扰动、库存回补和需求修复。
- 龙头企业价差、盈利弹性和现金流改善。

## 常见反向风险

- 价格回落、库存积累、需求走弱或加工价差收窄。
- 金价和工业品价格方向不同，行业判断很容易被写混。

## 首页应优先联想到的宏观 / 产业变量

- 资源品价格、库存、供给约束、基建与地产需求、出口与全球制造周期。

## 典型传导链

- 商品价格 / 供给格局变化 -> 企业毛利和现金流改善 -> 估值重估 -> 板块扩散或分化。

## 常见所处阶段

- 更常见的是“价格先行 -> 盈利预期修复 -> 业绩兑现 -> 高位分化”，而不是所有材料子行业同步上涨。

## 轮动与拥挤度

- 材料行业经常在黄金防守、有色顺周期、化工价差和建材修复之间切换；不要混成同一笔交易。

## 证伪信号

- 如果商品价格、库存和价差反向变化，或者需求端没有接力，这条行业逻辑就不能继续按顺风去写。

## 不能误写成直接催化的东西

- “材料涨价”不能自动写成当天催化。
- 商品价格上涨不能替代企业盈利和库存验证。

## 写首页时应避免的模板化表达

- 不要把黄金、有色、化工、建材写成一句“材料板块受益于顺周期”。
- 要说明今天到底是价格驱动、供给扰动，还是库存周期在起作用。
