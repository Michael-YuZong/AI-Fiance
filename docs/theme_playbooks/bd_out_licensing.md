# 创新药BD出海主题认知卡

## 市场通常在交易什么

- 市场在交易的是 BD 授权、海外合作、出海平台能力和全球估值映射。
- 这条线更强调“外部验证”而不是单纯国内估值修复。

## 常见正向驱动

- 大额 BD、海外合作、首付款和里程碑条款超预期。
- 龙头平台能力被外资药企或国际市场认可。

## 常见反向风险

- 预期很高，但后续新增 BD 节奏放缓。
- 市场把一次性事件高估成长期持续兑现。

## 首页应优先联想到的宏观 / 产业变量

- BD 节奏、海外合作、出海平台、估值映射、成长风格和港/A 医药风险偏好。

## 典型传导链

- 大额 BD / 出海验证 -> 龙头风险偏好修复 -> 创新药板块估值扩散。

## 常见所处阶段

- 常见于事件验证驱动的早中期修复阶段。

## 轮动与拥挤度

- 事件一出来就容易迅速抱团，后续最怕没有新增验证。

## 证伪信号

- 后续 BD 节奏没有延续，或者只是个别公司事件无法扩散到板块。

## 不能误写成直接催化的东西

- “创新药能出海”不能自动写成当天催化。
- 一次 BD 不能自动代表整个板块持续受益。

## 写首页时应避免的模板化表达

- 不要把 BD 出海和临床读数、CRO 写成同一个逻辑。
- 要点出这次市场交易的是外部验证，不只是估值修复。
