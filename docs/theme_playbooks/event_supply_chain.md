# 事件产业链主题认知卡

## 市场通常在交易什么

- 市场在交易的是大公司事件脉冲向供应链扩散，而不是产业链长期常识本身。
- 华为链、苹果链、英伟达链、特斯拉链都属于“核心公司新产品 / 财报 / 发布会 / 资本开支”带来的映射交易。
- 这类主题节奏快、脉冲强，但可持续性差异很大。

## 常见正向驱动

- 新品发布、财报超预期、订单验证、资本开支上修、供应链配套确认。
- 头部公司给出更清晰的产品周期、量产节奏或需求指引。
- 上游关键零部件或设备厂商被明确纳入受益链条。

## 常见反向风险

- 主题热度很高，但实际受益路径模糊。
- 事件驱动结束后，价格回归基本面。
- 市场把“链”越画越宽，最后真正受益者和跟风者混在一起。
- 一旦核心公司预期落空，整条链会快速退潮。

## 首页应优先联想到的宏观 / 产业变量

- 事件节点、财报、发布会、资本开支、订单与量产节奏。
- 这只标的在产业链里的位置，是直接受益还是二级映射。
- 当前是事件前预期交易、事件中兑现交易，还是事件后退潮。

## 典型传导链

- 核心公司事件节点 -> 订单 / 资本开支 / 备货预期抬升 -> 直接供应商先验证 -> 映射链条扩散 -> 兑现后再回到基本面筛选。

## 常见所处阶段

- 更常见的是“预热 -> 事件兑现 -> 直接受益确认 -> 跟风退潮”，而不是整条链持续同步上涨。

## 轮动与拥挤度

- 这类主题最容易先炒最宽泛的映射，再快速收敛到直接受益者；拥挤和回撤通常都比产业长期主线更快。

## 证伪信号

- 如果事件结果不及预期、供应链受益名单没被验证、资本开支和订单没有上修，或事件后价格明显转弱，这条主题就不能继续按主线扩散去写。

## 不能误写成直接催化的东西

- 只要贴上“某某链”标签，不能自动视为当天直接催化。
- 产业链想象不能替代订单、出货或业绩验证。
- 热度上升不能替代受益路径明确。

## 写首页时应避免的模板化表达

- 不要把“华为链 / 苹果链 / 英伟达链 / 特斯拉链”写成空标签。
- 要说明这波市场到底在交易新品周期、财报、capex，还是供应链份额提升。
- 不要忽略事件交易最怕的是兑现后回落。
