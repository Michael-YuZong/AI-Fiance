# 自动驾驶主题认知卡

## 市场通常在交易什么

- 市场在交易的是智驾功能升级、车路云配套、整车销量与渗透率提升带来的产业链弹性。
- 这条线更偏“功能兑现 + 政策基础设施 + 整车放量”的组合，不等同于机器人，也不等同于纯汽车整车。
- 当市场从“高弹性新质生产力”细分时，自动驾驶往往会和机器人、低空经济分家。

## 常见正向驱动

- 车企发布高阶智驾方案、FSD/NOA 路线推进、城市场景扩容。
- 车路云、政策试点、基建配套或激光雷达/线控底盘订单验证。
- 龙头整车和核心零部件厂商给出更明确的量产节奏。
- 市场风险偏好回到成长制造时，这条线容易重新获得溢价。

## 常见反向风险

- 题材热度高，但实际渗透率、盈利和订单兑现还慢。
- 整车销量承压时，智能化叙事容易被现实销量拖累。
- 从主题层看它常是高弹性方向，一旦市场切防守，回撤通常更快。
- 产业链验证并不均衡，整车、激光雷达、线控底盘和车路云并不会同步兑现。

## 首页应优先联想到的宏观 / 产业变量

- 智驾渗透率、整车销量、车路云政策进度、核心零部件订单验证。
- 成长制造风格是否顺风。
- 产业链究竟在交易“功能升级”，还是“整车销量 + 智能化双升”。

## 典型传导链

- 高阶智驾功能升级 / 政策试点 -> 整车与核心零部件渗透率预期上修 -> 订单与量产验证 -> 主题重新扩散。

## 常见所处阶段

- 更常处在“概念想象和量产验证并存”的过渡阶段，容易反复在预期和兑现之间摆动。

## 轮动与拥挤度

- 容易和机器人、低空经济、人形链一起被打包进高弹性成长；真正强的时候会从泛成长里走出独立智驾分支。

## 证伪信号

- 智驾渗透率、整车销量或政策试点节奏没有继续推进，且市场开始质疑兑现周期，说明这条线从预期驱动转向兑现压力。

## 不能误写成直接催化的东西

- “空间很大”“未来必然渗透”不能直接写成当天催化。
- 特斯拉、车企长期愿景不能直接替代本地产业链今天的订单或销量验证。
- 车路云政策大方向不能直接替代具体项目落地。

## 写首页时应避免的模板化表达

- 不要把自动驾驶重新写回“机器人与智能驾驶”一个大桶。
- 不要只写“新质生产力方向继续强”，要说明今天在交易智驾功能、车路云还是销量验证。
- 不要把政策想象直接写成可执行买点。
