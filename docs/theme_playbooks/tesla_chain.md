# 特斯拉链主题认知卡

## 市场通常在交易什么

- 市场在交易的是整车周期、FSD / Dojo / 机器人预期和供应链份额变化。
- 特斯拉链既能交易汽车销量，也能交易智能驾驶和新技术映射，所以波动来源比普通汽车链更复杂。

## 常见正向驱动

- FSD、Dojo、机器人预期强化，或车型周期、新平台和供应链份额提升。
- 智驾渗透、零部件升级和海外需求改善。

## 常见反向风险

- 主题容易在“智能驾驶想象”和“整车销量现实”之间反复摇摆。
- 如果销量或价格战压力偏大，事件逻辑会被基本面现实压制。

## 首页应优先联想到的宏观 / 产业变量

- FSD、Dojo、车型周期、销量、价格战、供应链份额、整车与零部件景气。

## 典型传导链

- FSD / 新车型 / 份额提升 -> 零部件与设备订单预期 -> 特斯拉链映射标的估值脉冲。

## 常见所处阶段

- 常在事件预期和销量验证交替主导的阶段运行。

## 轮动与拥挤度

- 一旦智驾主线活跃，特斯拉链很容易被快速抱团；但销量现实偏弱时回撤同样很快。

## 证伪信号

- FSD / 新平台预期没有继续强化，销量和价格战压力却持续恶化。

## 不能误写成直接催化的东西

- “特斯拉概念”不能自动写成当天催化。
- FSD 叙事不能替代订单、车型周期和销量验证。

## 写首页时应避免的模板化表达

- 不要把特斯拉链只写成自动驾驶链，也不要只写成汽车链。
- 要交代今天市场在交易智驾、机器人，还是整车销量与供应链份额。
