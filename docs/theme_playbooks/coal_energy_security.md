# 煤炭 / 能源安全主题认知卡

## 市场通常在交易什么

- 市场在交易的是能源安全、煤价韧性、供给约束和高现金流，而不只是“煤价涨跌”。
- 煤炭既有资源属性，也有高股息和现金流属性，经常处在“周期 + 防守”的交叉地带。
- 当能源安全重新升温时，市场通常先交易供给约束和现金流可见性，再看需求验证。

## 常见正向驱动

- 煤价高位维持、供给收缩、库存低位、迎峰度夏 / 迎峰度冬预期。
- 能源安全政策强化，火电煤电协同逻辑改善。
- 市场风格偏防守，高现金流和高股息方向重新获得承接。
- 进口煤扰动减弱，国内龙头盈利稳定性提高。

## 常见反向风险

- 煤价边际转弱，市场先交易现金流见顶。
- 成长风格回暖后，煤炭容易被视作旧经济而被资金减配。
- 高股息拥挤导致估值和赔率不再舒服。
- 能源安全逻辑仍在，但买点和趋势确认未必同步。

## 首页应优先联想到的宏观 / 产业变量

- 煤价、库存、火电需求、进口煤、能源安全政策、利率和分红率。
- 今天市场是在交易“资源价格”，还是“高股息防守”。
- 煤炭和原油 / 有色不是同一笔交易，要写清楚。

## 典型传导链

- 煤价韧性 / 供给约束 / 能源安全强化 -> 盈利和现金流可见性提高 -> 分红预期抬升 -> 板块估值获得支撑或修复。

## 常见所处阶段

- 更常见的是“资源价格支撑 -> 高股息承接 -> 估值修复 -> 拥挤后钝化”，而不是永远跟着商品价格线性上行。

## 轮动与拥挤度

- 煤炭常在高股息、公用事业、油气和电力之间轮动；成长风格强时它容易被边缘化，防守风格强时又容易快速抱团。

## 证伪信号

- 如果煤价、库存和火电需求同步转弱，分红预期被下修，或市场明确切回高弹性成长，这类首页就不能继续按顺风防守去写。

## 不能误写成直接催化的东西

- “能源安全重要”不能直接写成当天催化。
- 历史高股息不能自动替代当下趋势和位置。
- 煤价历史高位不能自动等于现在值得追。

## 写首页时应避免的模板化表达

- 不要把煤炭直接写成“资源股受益于商品上涨”。
- 不要忽略它同时带有高股息、防守和周期三层属性。
- 不要把能源安全口号写成模糊利好。
