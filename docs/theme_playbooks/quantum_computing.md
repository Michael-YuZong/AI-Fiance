# 量子计算主题认知卡

## 市场通常在交易什么

- 市场在交易的是前沿科技叙事、政策支持和科研突破映射，而不是短期商业化兑现。
- 量子计算、量子通信和量子测量常被一起交易，但商业路径和产业成熟度差异很大。

## 常见正向驱动

- 科研突破、政策支持、重点实验室和产业合作进展。
- 市场风险偏好提升，愿意交易高想象力的科技主题。

## 常见反向风险

- 商业化路径很远，基本面和业绩映射极弱。
- 主题很容易变成纯情绪脉冲，持续性差。

## 首页应优先联想到的宏观 / 产业变量

- 政策、科研节点、技术突破、产业化合作、风险偏好和成长风格。

## 典型传导链

- 科研 / 政策节点 -> 主题情绪升温 -> 映射标的估值脉冲。

## 常见所处阶段

- 更像事件脉冲和早期前沿技术映射阶段。

## 轮动与拥挤度

- 典型的高弹性高波动主题，容易在科技题材活跃时被情绪迅速放大。

## 证伪信号

- 科研节点没有新增进展，或者市场风格迅速切回防守。

## 不能误写成直接催化的东西

- “前沿科技空间大”不能直接写成当天催化。
- 科研概念不能直接替代收入和利润验证。

## 写首页时应避免的模板化表达

- 不要把量子计算写成已经进入产业兑现阶段。
- 不要把政策和想象力误写成基本面确认。
