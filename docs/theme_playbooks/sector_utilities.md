# 公用事业行业认知卡

## 市场通常在交易什么

- 市场在交易的是稳现金流、监管定价、资本开支和防守属性，而不是一句泛化的“公用事业稳”。
- 电力、水务、燃气等公用事业同属防守类资产，但对燃料成本、投资周期和分红的敏感度不同。

## 常见正向驱动

- 利率回落、防守风格升温、分红回报稳定、监管机制改善和资本开支更可见。
- 燃料成本回落或盈利稳定性增强。

## 常见反向风险

- 拥挤后估值并不便宜，防守资产也会回撤。
- 成长风格回暖时，公用事业容易被资金减配。

## 首页应优先联想到的宏观 / 产业变量

- 利率、分红率、燃料成本、监管定价、资本开支和风格偏好。

## 典型传导链

- 利率 / 风格 / 定价机制改善 -> 现金流与分红可见性提升 -> 估值修复 -> 防守资金承接。

## 常见所处阶段

- 更常见的是“防守承接 -> 估值修复 -> 抱团拥挤 -> 分化”，而不是所有时候都适合追高。

## 轮动与拥挤度

- 公用事业常和红利、银行、煤炭等防守资产一起轮动；一旦抱团过深，回撤速度也会明显加快。

## 证伪信号

- 如果利率重新上行、分红吸引力下降或风格切回高弹性成长，这条行业逻辑就不能继续按舒服防守去写。

## 不能误写成直接催化的东西

- “公用事业高股息”不能自动写成当天催化。
- 稳定现金流不能替代趋势位置和拥挤度判断。

## 写首页时应避免的模板化表达

- 不要只写“公用事业适合震荡市配置”。
- 要说明今天交易的是分红、防守、成本回落，还是投资周期改善。
