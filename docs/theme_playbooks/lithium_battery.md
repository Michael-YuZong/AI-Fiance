# 锂电主题认知卡

## 市场通常在交易什么

- 市场在交易的是电动车需求、材料价格、产业链库存和龙头盈利修复。
- 锂电主链更偏产业链周期和材料价格，不同于固态电池这类远期技术路线。

## 常见正向驱动

- 电动车销量改善、去库加快、正负极 / 电解液 / 隔膜景气回暖。
- 上游价格企稳，龙头公司盈利能力和资本开支预期改善。

## 常见反向风险

- 需求边际改善不足，价格战和库存继续压制利润率。
- 市场把新技术想象和主链盈利修复混成一个故事。

## 首页应优先联想到的宏观 / 产业变量

- 电动车销量、材料价格、库存、出海、龙头产能利用率和利润率。

## 典型传导链

- 终端销量 / 去库改善 -> 材料与电池环节盈利预期回稳 -> 锂电链估值修复。

## 常见所处阶段

- 常见于库存周期后段的修复预期阶段。

## 轮动与拥挤度

- 一旦销量与去库信号共振，锂电容易从低位修复迅速转成拥挤交易。

## 证伪信号

- 销量不及预期、材料价格继续下行、库存迟迟不去。

## 不能误写成直接催化的东西

- “电动车渗透率长期提升”不能直接写成当天催化。
- 技术迭代叙事不能替代主链订单和盈利修复。

## 写首页时应避免的模板化表达

- 不要把锂电主链和固态电池写成同一种逻辑。
- 不要只写“新能源车产业链景气向上”。
