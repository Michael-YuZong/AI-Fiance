# 钙钛矿主题认知卡

## 市场通常在交易什么

- 市场在交易的是光伏新技术路线的效率提升、降本潜力和中试量产进展。
- 钙钛矿常被当成“下一代光伏效率革命”来交易，但现实里更多还是设备、工艺和示范线验证。

## 常见正向驱动

- 中试线、叠层电池效率突破、头部厂商示范项目推进。
- 设备和材料环节出现更明确的订单或验证节奏。

## 常见反向风险

- 技术路线仍远，真正大规模量产和成本优势尚未验证。
- 光伏主产业链景气低迷时，新技术主题更容易被当成远期叙事。

## 首页应优先联想到的宏观 / 产业变量

- 效率突破、中试线、蒸镀 / 涂布设备、量产节奏、光伏景气和政策方向。

## 典型传导链

- 效率 / 中试验证 -> 设备与材料环节订单预期 -> 新技术路线估值溢价。

## 常见所处阶段

- 更像技术验证和中试预期阶段，而不是成熟景气主线。

## 轮动与拥挤度

- 常在光伏反弹或科技制造主题扩散时被短线放大，拥挤度来得快也退得快。

## 证伪信号

- 关键效率或量产节点迟迟不落地，或者光伏主链景气继续恶化。

## 不能误写成直接催化的东西

- “下一代光伏技术”不能直接写成当天催化。
- 技术突破新闻不能替代订单和量产验证。

## 写首页时应避免的模板化表达

- 不要把钙钛矿写成已经能替代主流光伏路线。
- 不要把远期技术故事写成当前业绩逻辑。
