# AI算力主题认知卡

## 市场通常在交易什么

- 市场在交易的是 AI 资本开支和算力基础设施的持续性，而不是单一概念名称。
- 核心链条通常包括光模块、服务器、交换机、液冷、IDC、电源和上游核心芯片。
- 这类方向最容易在“需求强 + 估值高 + 波动大”三件事里同时出现。

## 常见正向驱动

- 海外云厂商和互联网平台继续上调 AI capex。
- 服务器、光模块、交换机和液冷订单继续验证。
- 大模型迭代、推理需求扩张，使算力从一次性投资变成持续扩容故事。
- 国产替代或本地供应链份额提升带来新的弹性。

## 常见反向风险

- 订单和业绩没有变坏，但估值先走得太快。
- 海外 capex 增速边际放缓，市场开始怀疑高增长能否持续。
- 板块内部从“全面扩散”切回“只剩龙头”，中后排标的容易掉队。
- 资金热度极高时，很容易先交易拥挤，再交易兑现。

## 首页应优先联想到的宏观 / 产业变量

- 海外云厂商 capex、AI 服务器出货、光模块需求、交换机升级、液冷渗透率。
- 成长风格顺逆风、利率和 risk-on/risk-off 是否支持高估值资产。
- 算力是仍在主升段，还是进入高位分化与验证阶段。

## 典型传导链

- 海外云厂商 capex -> 服务器 / 光模块 / 交换机 / 液冷订单 -> 产业链景气扩散 -> 高估值成长溢价。

## 常见所处阶段

- 常在“高景气 + 高估值 + 高波动”并存阶段运行，后期更容易进入高位分化。

## 轮动与拥挤度

- 最怕的是主线拥挤后只剩龙头，或者 capex 叙事边际减弱时资金先撤中后排。

## 证伪信号

- 海外 capex 边际放缓、订单验证转弱、份额和成交热度背离，说明这条线开始从景气交易退化成纯拥挤交易。

## 不能误写成直接催化的东西

- “AI 是长期大趋势”不能直接写成当天催化。
- 海外巨头长期看好 AI 不能直接等于这只标的今天就能做。
- 讨论度和题材热度不能直接替代订单、业绩和价格确认。

## 写首页时应避免的模板化表达

- 不要只写“AI 方向长期看好”。
- 不要把“算力”写成抽象名词，要点出市场在交易哪条子链。
- 不要忽略高估值和高拥挤度是这类标的最大的短线风险。
