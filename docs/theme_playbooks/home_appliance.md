# 家电主题认知卡

## 市场通常在交易什么

- 市场在交易的是地产后周期、以旧换新、出口订单和龙头盈利韧性。
- 家电更像“内需 + 出口 + 制造龙头”的复合逻辑，不只是消费修复。

## 常见正向驱动

- 以旧换新、地产后周期改善、出口订单修复、原材料成本回落。
- 龙头公司经营质量和现金流更稳，带来估值修复。

## 常见反向风险

- 地产后周期和内需改善不及预期。
- 出口承压或汇率扰动影响利润率。

## 首页应优先联想到的宏观 / 产业变量

- 地产后周期、以旧换新、出口、原材料成本、白电/小家电景气差异。

## 典型传导链

- 地产 / 以旧换新 / 出口修复 -> 龙头订单与利润率改善 -> 家电板块估值修复。

## 常见所处阶段

- 常见于低位修复和盈利预期改善阶段。

## 轮动与拥挤度

- 更偏中大盘机构风格，拥挤度通常低于白酒和纯题材消费。

## 证伪信号

- 内需和地产后周期没有改善，出口和利润率同步走弱。

## 不能误写成直接催化的东西

- “消费复苏”不能自动等于家电主线确认。
- 以旧换新预期不能替代销量和利润率验证。

## 写首页时应避免的模板化表达

- 不要把家电写成和白酒、免税一样的消费修复逻辑。
- 要交代今天更偏内需、出口，还是地产后周期。
