# /assistant

## 输入

- 一句自然语言请求

## 执行流程

1. 识别请求里的意图、标的、规则、场景和时间区间
2. 若用户说的是中文 ETF 名称或主题名，先尝试解析成代码
3. 自动路由到最匹配的命令，例如 `briefing`、`scan`、`risk`、`backtest`、`lookup`
4. 如果无法稳定匹配，回退到 `research`

## 输出

- 路由结果
- 实际执行的命令
- 对应命令正文

## 适用场景

- 用户不会记命令
- 在 Codex / Claude Code 里直接说“帮我做什么”
- 用户只知道“有色金属ETF”“黄金ETF”这类中文名字
