# Skill Command Map

这份文件只做一件事：告诉你 `skill/commands/` 里每份命令说明什么时候读。

默认不要把这个目录所有命令文档一起读完。

## 默认读法

| 场景 | 先读哪份 |
| --- | --- |
| 自然语言入口 | [assistant.md](./assistant.md) |
| 市场主线 / 晨报 / 周报 | [briefing.md](./briefing.md) |
| 单标的深度分析 | [scan.md](./scan.md) |
| 机会发现 / 主题扫描 | [discover.md](./discover.md) |
| 同类对比 | [compare.md](./compare.md) |
| 标的名称找代码 | [lookup.md](./lookup.md) |
| 研究问答 | [research.md](./research.md) |
| 组合与交易记录 | [portfolio.md](./portfolio.md) |
| 风险报告 / 压测 | [risk.md](./risk.md) |
| 回测规则 | [backtest.md](./backtest.md) |
| 宏观 regime | [regime.md](./regime.md) |
| 政策解读 | [policy.md](./policy.md) |
| 盘中快照 | [snap.md](./snap.md) |

## 最常见最小集合

- 大多数任务：`assistant / briefing / scan / discover`
- 用户明确要命令细节时：只打开对应那一份
- 需要背景规则时：再看 [../references/README.md](../references/README.md)
