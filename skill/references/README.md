# Skill Reference Map

这份文件只做一件事：告诉你 `skill/references/` 里每份参考文档是什么，什么时候才需要打开。

默认不要把这个目录所有参考文档都读一遍。

## 路由

| 文件 | 什么时候看 |
| --- | --- |
| [briefing-quality.md](./briefing-quality.md) | 需要生成正式 `briefing daily`，或检查晨报是否像投顾成稿 |
| [markdown-review-loop.md](./markdown-review-loop.md) | 当前任务会产出独立 Markdown 成稿，需要走外审闭环 |
| [tushare-a-share.md](./tushare-a-share.md) | A 股数据口径、接口选型、字段/单位校验不稳时 |

## 默认顺序

1. 先看 [../SKILL.md](../SKILL.md)
2. 再看 [../commands/README.md](../commands/README.md)
3. 只有被场景命中时，才打开这里的具体参考文档
